🍣 Sushiya – Sushi Restaurant Website

> A modern, responsive sushi restaurant landing page built with clean UI and smooth user experience.

🌐 **Live Demo:** https://abrvion.github.io/Sushiya-Project/



 ✨ Overview

Sushiya is a visually appealing restaurant website designed to showcase sushi dishes, menus, and a premium dining experience.

It focuses on:
- Clean layout 🎨
- Smooth user experience ⚡
- Mobile responsiveness 📱



 🚀 Features

- 🍱 Beautiful sushi menu showcase  
- 📱 Fully responsive design  
- ⚡ Fast and lightweight  
- 🎨 Modern UI/UX design  
- 🧭 Easy navigation  



## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & layout  
- **JavaScript** – Interactivity
